# 工业AI交付方案包 · 客户提案

## 痛点分析

工业客户有三大痛点：
1. **合规风险**：AI辅助控制操作是否违反安全标准？谁来负责？
2. **调参门槛**：MIMO回路调参靠老师傅经验，缺人时怎么办？
3. **因果黑盒**：出了问题不知道根因，只知道报警

## 我们的方案

一套工具解决三个痛点：

| 模块 | 解决问题 | 输出 |
|------|---------|------|
| 合规审查 | AI操作是否安全合规 | 合规报告（PASS/WARN/FAIL） |
| MIMO调参 | 控制器参数怎么整定 | PID参数 + 配对建议 |
| 因果发现 | 变量之间谁是因谁是果 | 因果图 + 根因分析 |

## 交付流程

```
需求确认 → 数据采集 → 三模块执行 → 交付报告
    1天        2-3天       1-2天       1天
```

## 案例

### 案例一：压缩机站MIMO控制优化
- 2输入2输出，变量：进气压力、排气温度、流量
- 合规审查：100%通过
- MIMO调参：ZN Kc=12.0, Ti=0.2; Cohen-Coon Kc=10.3, Ti=0.28
- 因果发现：进气压力+排气温度→流量（对撞结构）

### 案例二：注塑成型过程优化
- 3变量：熔体温度、注射压力、产品缺陷率
- 合规审查：100%通过
- MIMO调参：ZN Kc=12.0, Ti=0.1; Cohen-Coon Kc=10.3, Ti=0.14
- 因果发现：熔体温度+注射压力→缺陷率（对撞结构）
