# 工业AI交付方案包 v1.0

## 方案概述

**目标客户**: 流程化工、离散制造企业，有PLC/SCADA系统但缺AI能力

**核心卖点**: 一套工具解决三个痛点——合规审查、控制器调参、因果分析

**交付周期**: 1-2周（标准场景）

## 三模块

### 模块一: 工业AI合规审查
- 基于 IEC 61508 / ISO 13849
- 8条硬规则，自动检测违规
- 输出: 合规报告（PASS/WARNING/VIOLATION）

### 模块二: MIMO控制器调参
- ZN整定 + Cohen-Coon + Lambda
- RGA配对分析
- 静态解耦器设计
- 输出: PID参数 + 配对建议

### 模块三: 因果发现
- PC算法标准实现
- Fisher Z检验
- v-结构识别
- 输出: 因果图 + 模式分类

## 交付流程

1. 需求确认 → 2. 数据采集 → 3. 三模块执行 → 4. 报告交付
